Projekt

Coop Azure datapipeline

Problem
Betalnings- och transaktionsdata behövde ett stabilt och automatiserat flöde för att möjliggöra analys, kvalitetskontroller och verksamhetsrapportering utan manuella steg.
Lösning
En Azure baserad datapipeline byggdes för att automatiskt hämta transaktionsfiler, lagra data i Blob Storage och bearbeta den vidare till Azure SQL och Power BI. Lösningen skapade ett mer tillförlitligt och automatiserat flöde för analys och rapportering.
Tekniker
Azure Data Factory, Azure SQL database, Storage Account, Power BI och SQL.
Min roll
Satt upp Azure resurser och utvecklade den automatiserade datapipelinen i Azure Data Factory, inklusive integration mot FTP-broker, lagring i Storage, databearbetning i Azure SQL samt underlag för visualisering i Power BI. Arbetade även med datamodellering, kvalitetskontroller och automatisering för att möjliggöra ett stabilt och automatiserat flöde utan manuella steg. Arbetade även med felsökning, datavalidering och samarbete mellan verksamhet och teknik för att säkerställa stabila flöden.

AI-kunskapscopilot / AI-agent

Problem
Anställda i Coops interna IDA-plattform behövde snabbare stöd för att hitta instruktioner och hantera accessrelaterade ärenden, exempelvis behörigheter, systemåtkomst och administrativa processer.
Lösning
Utvecklade en AI-driven kunskapscopilot som kombinerar RAG (Retrieval-Augmented Generation) med interna API:er för att ge kontextbaserade svar och stöd i ärendeflöden. Lösningen integrerades med Azure AI Search, Azure OpenAI och Azure Functions för att möjliggöra både kunskapssökning och användarspecifik accessinformation.
Tekniker
Azure AI Search, Azure OpenAI, Azure AI Foundry, Azure Functions, RAG, OpenAPI, REST API:er, Storage Account, Vector Search.
Min roll
Designade och konfigurerade en PoC för en AI-agent i Azure AI Foundry, inklusive kunskapsbas med RAG, indexering i Azure AI Search och verktygsintegration via OpenAPI. Utvecklade och deployade Azure Functions för användarspecifika accessfrågor samt definierade agentlogik och instruktioner för säkra och relevanta svar. Lösningen verifierades end-to-end från användarfråga till API-anrop och svar.
